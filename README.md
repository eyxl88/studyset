## Project Description
Allows users to create, save, and add to study sets through the terminal. Provides quizzing features in multiple choice mode (answer with term or definition), select all mode, match mode, order chronologically / logically, and type the key term. Also offers flashcard like flipping. 

## Features
- Quiz mode
- Keywords
- And more!
